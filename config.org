* EShell
This binds Alt + r to a history popup using counsel.

#+begin_src emacs-lisp
(after! eshell
  (add-hook 'eshell-mode-hook
            (lambda ()
              (eshell-cmpl-initialize)
              (define-key eshell-mode-map (kbd "M-r") 'counsel-esh-history)))
)
#+end_src
